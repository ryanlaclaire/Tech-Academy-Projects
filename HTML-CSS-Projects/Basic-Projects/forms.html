Tools like Dreamweaver are great, especially if you are building old-style web sites based on tables, sliced images, etc. But if you want to be able to build modern looking websites, then these tools will get in the way of you learning and will actually slow you down!

CSS Layout Types

  • Static Design

    • - Does not change. The size stays the same regardless of browser size. It's not flexible.

  • Liquid Design

    • - Also flows and flexes but will not resize items like images, search box, button placement.

  • Adaptive Design

    • Produces a site that is fixed pixel with some elements of responsive.

  • Responsive Design

    • - It flows and changes based on the size of the window. The most flexible. Looks good on any device. parts of the page will resize
